## Installing for developpement
With conda
```
git clone https://github.com/Chthi/ecosysteme
cd ecosysteme
conda env create --file requirements.txt -n ecosysteme
conda activate ecosysteme
pip install --editable ./
```

## Usage
```
python -m ecosysteme
```
